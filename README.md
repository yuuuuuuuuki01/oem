# 酒蔵OEM見積ツール

日本酒酒蔵のOEM受託案件向けに、仕込み設計から見積金額までを一気に試算する静的Webツールです。GitHub Pages でそのまま公開できます。

## できること

- `出荷予定量` を加水後の最終液量として扱う
- 出荷予定量に対して歩留まりロスを見込み、必要製造量を自動で上乗せする
- 目標アルコール度数と各酒類の度数・純アルコール寄与率から必要量を逆算する
- 玄米単価、精米歩合、白米1kgあたり製成酒量から原酒由来コストを算出する
- 原料用アルコール、ブレンド酒、果汁などの必要量と原価を反映する
- 果汁など副原料の加工費を固定費として別建て計上できる
- 容量帯ごとの包装単価を5帯まで設定し、瓶サイズに応じて自動適用する
- 醸造期間、人員数、人件費から製造人件費を算出する
- 設備・ユーティリティ費を `円/L・原酒` で原酒量に応じて反映する
- 酒類区分と推定アルコール度数から酒税を自動計算する
- 包装費、固定費、利益率、消費税率を含めて見積総額を算出する
- CSV出力、印刷、入力内容の `localStorage` 保存に対応する

## 酒税について

- 画面上の `酒類区分` で `清酒 / リキュール / 果実酒 / スピリッツ` を選択できます。
- 酒税は国税庁の酒税率一覧表をもとにした簡易計算です。
- このツールでは、最終的な配合後の推定アルコール度数と酒類区分から `1kl あたり税率` を求め、出荷想定量に乗じています。
- 実際の課税判定は原料、製法、免許区分、法令解釈で変わるため、正式見積や出荷前には必ず最新の法令・所轄税務署運用で確認してください。

## 使い方

1. `index.html` をブラウザで開きます。
2. `案件条件` で酒類区分、容量、歩留まり、出荷予定量を入力します。
3. `配合・酒税条件` で原酒度数、各酒類の純アルコール寄与率、果汁等の比率を入力します。
4. `原価条件` と `包装・固定費条件` を入力します。
5. 右側の `見積内訳` と `見積プレビュー` を確認します。
6. 必要に応じて `CSV出力` または `印刷` を使います。

## GitHub Pages

このリポジトリには GitHub Pages 用 workflow を追加してあります。

1. GitHub のリポジトリ設定で `Settings > Pages` を開きます。
2. `Build and deployment` の `Source` を `GitHub Actions` にします。
3. `main` ブランチへ push すると `Deploy GitHub Pages` workflow が走ります。
4. 公開URLは通常 `https://yuuuuuuuuki01.github.io/oem/` です。

## ファイル構成

- `index.html`: UI
- `style.css`: レイアウトとスタイル
- `app.js`: 計算ロジックと画面更新
- `.github/workflows/deploy-pages.yml`: GitHub Pages デプロイ

## 補足

- 依存パッケージは使っていないため、静的ホスティングでそのまま動きます。
- 入力値はブラウザの `localStorage` に保存されます。
